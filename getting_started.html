Getting started

Installation

pip

TorchLaplace can be installed via pip from PyPI.

pip install torchlaplace

Tutorials

For a quick overview of TorchLaplace functionality, see the user guides.

You can also refer to the reference api for an exhaustive list of all what is possible with torchlaplace.

NeuralLaplace Experiments

To replicate the experiments from Neural Laplace, you will have to install additional packages.

pip

TorchLaplace additional packages for running experiments from Neural Laplace can be installed via pip from PyPI.

pip install torchlaplace[all]
